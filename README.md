## 安装

macOS Catalina (10.15) and later


| # | Step                                                             |
| --- | ------------------------------------------------------------------ |
| 1 | Install xcode via apple store                                    |
| 2 | git clone https://github.com/mtjszqjgmailcom/chargeapp.git <br /> cd chargeapp <br /> npm i |
| 3 | curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs\| sh   |
| 4 | source $HOME/.cargo/env                                          |
| 5 | npm run tauri dev                                                |
